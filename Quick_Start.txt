## 🔍 Debug Checklist

### If Project Not Appearing:
```bash
1. Open Browser Console (F12)
2. Check for errors (red text)
3. In component, add:
   console.log('Projects:', projects);
   console.log('Filtered:', filteredProjects);
4. Check curationStatus value
5. Check filter conditions
```

### If Login Not Working:
```bash
1. Check email exact match (case sensitive)
2. Check password = "123" (not 123)
3. Check browser console for errors
4. Try different account
```

### If Data Not Saving:
```bash
1. Check browser console
2. Verify updateProject called
3. Check ProjectsContext provider
4. Check component is inside provider
5. Refresh page (mock data resets!)
```

### 2. Viewing Mock Data
```bash
Check these files for current data:

- /components/ProjectsContext.tsx (projects)
```

### 3. Reset Data
```bash

(Mock data resets to initial state)
```

### 4. Check Integration
```bash
Look for:
✅ Toast notifications (success/error)
✅ Counter updates
✅ Data appearing in next step
✅ Filtered lists showing correct items
```
